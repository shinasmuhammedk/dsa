# Go Data Structures and Algorithms
This repository contains implementations of common Data Structures
and Algorithms written in Go. It is created for learning, practice,
and interview preparation.

## Topics Covered

- Sorting Algorithms
  - Bubble Sort
  - Insertion Sort
  - Selection Sort

- Data Structures
  - Binary Search Tree
  - Heap (Min Heap / Max Heap)

- Hash Map Problems
  - Two Sum
  - Word Frequency Counter
  - Duplicate Number Detection

- String Algorithms
  - Reverse String
  - Palindrome Check

- LeetCode Problems
  - Group Anagrams
